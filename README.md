# MiniStore

MiniStore is a full-stack e-commerce web app built with **Flask**, **MySQL**, and **JavaScript**. Users can browse products, add items to the cart, and place orders. Admins can manage products, view orders, and update order statuses efficiently.

## Features
- User registration and login with secure password hashing
- Product listing, addition, and deletion
- Shopping cart and checkout system
- Order placement and tracking
- Admin dashboard for managing products and orders
- Responsive design for desktop and mobile

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, Flask
- **Database:** MySQL
- **Libraries & Tools:** bcrypt, Flask-CORS, Werkzeug
